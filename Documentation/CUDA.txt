Cara aktivasi CUDA agar streaming berjalan lancar 
==================================================
1) onnx for FR
$ pip install onnxruntime-gpu==1.20.1

2) install CUDNN
conda : 

$ conda install nvidia::cudnn cuda-version=12

pip :

$ pip install nvidia-cudnn

3) Cek path cudart

$ where cudart64*.dll

hasil yang diharapkan :

C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v12.8\bin\cudart64_12.dll
C:\Program Files (x86)\NVIDIA Corporation\PhysX\Common\cudart64_65.dll
